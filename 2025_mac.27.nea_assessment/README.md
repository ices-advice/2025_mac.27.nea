# 2025_mac.27.nea_assessment
2025 - Mackerel (Scomber scombrus) in subareas 1-8 and 14 and in divisions 9.a, 12.a, and 12.b (Northeast Atlantic and adjacent waters) - WGWIDE (ADGWIDE)

## Naming convention for the data prep
Data script      | Description
-----------|--------------
**Models 0.x**| **Data revisions**
data0.00.r     | 	reproducing the data for the accepted assessment at WKBMACNSSH 2025.
data0.01.r     | 	loading the updated data for the 2025 WGWIDE assessment.
data0.02.r     | 	as data0.01.r without the 2025 MEGS point

## Naming convention for the model runs

Model      | Description
-----------|--------------
**Models 0.x**| **Data revisions**
model0.00     | 	reproducing the accepted assessment at WKBMACNSSH 2025.
model0.01     | 	model using the new 2025 WGWIDE data.
model0.02     | 	as model 0.01, removing 2025 MEGS point
